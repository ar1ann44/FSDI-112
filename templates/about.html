About This Project

A Django web application created for book lovers and readers

Home Page

The Home page is the main section of the website, here users can explore popular books and discover new stories through a simple design

Contact Page

The Contact page contains personal contact information and allows visitors to get in touch with the creator of the project

Post Page

The Post page displays all published book reviews. Users can read review titles, short descriptions, and click on “Continue reading...” to view the full review and see more about each book

New Post Page

The New Post page allows users to create a new book review. Users simply enter a title, subtitle, and description in the form, then click the create button to publish their post
